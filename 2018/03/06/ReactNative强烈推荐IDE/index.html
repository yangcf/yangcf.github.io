ReactNative强烈推荐IDE

发表于 2018-03-06

在RN开发过程中先后使用了好几个常见的IDE：IDEA、WS、Android studio等，最后觉得最好用的还是VScode。
强烈推荐！
她是一个非常轻量级的编辑器，而且有强大的插件支持，几乎你能想到的功能都可以找到对应的拓展插件，十分便捷，现在推荐一些开发RN必要的插件。

  1. Babel ES6/ES7 必要！
    检查ES6/ES7的语法，并且会高亮，很漂亮，必须安装（别告诉我，你还在用ES5）。
  2. React Native Tools 必要！
    这个是微软出的开发工具，非常好用，让你摆脱命令行的束缚。
  3. ESLint
    代码检查工具，可以通过修改.eslintrc文件进行制定，团队开发有必要了解，有利于代码风格的统一。
  4. Path Intellisense
    路径补全插件，可以快速定位import的模块路径，必要！
  5. Typing auto installer
    自动根据package.json安装对应模块，再也不用npm install了
  6. Beautify
    代码美化插件，不是必须，但是真的不错。
  7. GitLens/tortoise-svn
    版本控制git和svn，这个不多说了。
